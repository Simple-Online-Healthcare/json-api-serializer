<h1 align="center">json-api-serializer</h1>

# Introduction

This package allows for easy serialization of Doctrine entities to JSON:API.

# Installation

1. Add the repository to your `composer.json`
    - `composer config repositories.simple-online-healthcare/json-api-serializer vcs https://github.com/Simple-Online-Healthcare/json-api-serializer`
2. Require the package within your application
    - `composer require simple-online-healthcare/json-api-serializer`
3. Laravel should auto-discover the package.
